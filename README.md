## how to use
1.
```
git clone https://github.com/null8r/ubuntu-setup.git
```

2.
```
cd ubuntu-setup
```

3.
```
sh setup.sh
```
or
```
./setup.sh
```

<br>
<br>

```
git clone https://github.com/null8r/ubuntu-setup.git && cd ubuntu-setup && sh setup.sh ; ./setup.sh
```
